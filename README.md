# Cash Flow Management Project

## Project Overview
The Cash Flow Management Project is a C# application designed to help individuals and businesses track and manage their cash flow. It provides users with tools to monitor income, expenses, and provide insights into financial health over time.

## Features
- User authentication and profile management
- Add, edit, and delete income and expense entries
- Generate reports by date, category, and custom filters
- Visual representation of cash flow trends using charts
- Export data to CSV for external use
- Built-in guidelines and tips for effective cash flow management

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/PhSM-PhEN/CashFlowProject.git
   ```
2. Navigate to the project directory:
   ```bash
   cd CashFlowProject
   ```
3. Open the solution file in Visual Studio.
4. Restore the NuGet packages:
   ```bash
   dotnet restore
   ```
5. Build the project:
   ```bash
   dotnet build
   ```
6. Run the application:
   ```bash
   dotnet run
   ```

## Usage
After setting up the application, users can sign in or create a new account. The interface allows users to record transactions, view current cash flow status, and generate financial reports. Users can filter records by date and category to analyze their spending patterns.
